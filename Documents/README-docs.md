# Docs index — Scripts documentation

Tài liệu này là index cho các file Markdown đã tạo nhằm giúp bạn nhanh nắm và mở rộng codebase.

Các tài liệu chính (tạo sẵn):
- `Scripts-Overview.md` — Tổng quan folder `Assets/_MonstersOut/Scripts`.
- `AI.md` — Mô tả folder `AI` và file chính như `Enemy.cs`, `SmartEnemyGrounded`.
- `Controllers.md` — Raycast controller, Controller2D, Projectiles.
- `Helpers.md` — Các helper (GlobalValue, SpawnItemHelper, AnimationHelper, v.v.).
- `Managers.md` — GameManager, LevelManager, Shop, SoundManager.
- `Player.md` — Tổng quan folder Player.
- `UI.md` — Các script UI và cách chúng kết nối với GameManager.
- `Roadmap.md` — Lộ trình học để hiểu và phát triển project.
- `Unity-Concepts.md` — Khái niệm Unity cần biết (ngắn gọn).
- `Workflow-Tasks.md` — Quy trình làm task, debug tips, kiểm tra trước khi merge.
- `ShopUI.md` — Flow mua hàng local & IAP.
- `Map.md` — Cách map/level được load và cấu hình level prefab.
- `Character-Properties.md` — Chi tiết `UpgradedCharacterParameter` và PlayerPrefs.
- `Events-and-Triggers.md` — IListener, animation events, projectile collisions, UI button hooks.
- `Player-Deep.md` — Deep dive `Player_Archer` (trajectory, shooting logic).
- `Enemy-Deep.md` — Deep dive `Enemy` base, attack modules, effects.
- `Core-Objects.md` — Camera, Sound, Menu, GameManager explanation.
- `Namespaces.md` — Vì sao dùng `namespace RGame` và cách dùng namespace.

Hướng dùng nhanh:
1. Mở `Roadmap.md` để thấy thứ tự học.
2. Mở `Scripts-Overview.md` rồi chọn folder bạn muốn tìm hiểu (ví dụ `AI.md` hoặc `Player-Deep.md`).
3. Thực hành theo `First-Tasks.md` (file kế tiếp) để bắt tay sửa code.
